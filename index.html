About Me

I'm a self-taught developer based in Nairobi, Kenya, working through the complete FreeCodeCamp Full Stack curriculum. My goal is to build real, useful software — and land my first freelance clients through Upwork and Fiverr in the process.

The curriculum covers six tracks: Responsive Web Design, JavaScript Algorithms, Front-End Libraries (React), Back-End Development and APIs, Relational Databases, and Scientific Computing with Python. Every project in this portfolio comes from that curriculum — real working code, pushed to GitHub.

I'm currently available for small to medium freelance projects — particularly landing pages, Python automation scripts, web app components, and data tasks. I work remotely and communicate clearly throughout every project.
